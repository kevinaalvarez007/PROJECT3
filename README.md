# Mercado Energético Colombiano: Análisis y Predicción de Precios

Este repositorio contiene el desarrollo del proyecto “Mercado Energético Colombiano: Análisis y Predicción de Precios”, enfocado en el estudio del comportamiento del mercado eléctrico colombiano y la estimación de precios mediante análisis de datos y modelos predictivos.

---

## Contenido del Repositorio

| Archivo | Descripción |
|----------|--------------|
| Mercado_Energético_Colombiano_Análisis_y_Predicción | Documento principal del trabajo académico con el análisis, resultados y conclusiones. |
| Predicción_mercado_energético.ipynb | Notebook de Google Colab que contiene el código para la limpieza, análisis exploratorio y modelado predictivo del mercado energético. |
| european_wholesale_electricity_price_data_daily.csv | Conjunto de datos de precios eléctricos diarios, utilizado como referencia comparativa con el mercado europeo. |
| README.md | Archivo de descripción del proyecto y guía de uso del repositorio. |

---

## Descripción del Proyecto

El objetivo principal es analizar el comportamiento histórico del precio ponderado de la energía en Colombia durante el año 2024 y desarrollar un modelo predictivo que permita estimar su tendencia futura.  
Se aplican técnicas de análisis de datos y métodos numéricos implementados en Python.

## Ejecución

1. Abrir el archivo *Predicción_mercado_energético.ipynb* directamente en *Google Colab* utilizando el enlace del repositorio.  
2. Seguir el flujo de ejecución de las celdas en orden.  
3. Cuando Colab lo solicite, subir el archivo *european_wholesale_electricity_price_data_daily.csv* desde el equipo local.  
4. Una vez cargado el archivo, continuar con la ejecución normal para visualizar los resultados y las gráficas.
